# Anki 安装、导入与学习指南

本文记录当前电脑上已经完成的 Anki 配置，以及以后导入新卡组时的操作方法。

## 当前已完成的配置

- 已安装官方 Windows 版 Anki 26.05；
- 桌面入口：`C:\Users\Administrator\Desktop\Anki.lnk`；
- 已导入卡组：`英语四级乱序词汇 · 已生成音频`；
- 卡组包含 1104 张单词卡和 1104 个 WAV 音频；
- 每日新卡片上限：30 张；
- 每日复习上限：200 次；
- 自动播放音频：开启；
- 导入时没有导入旧学习进度，当前卡片均从新卡开始学习。

当前导入源文件保存在：

```text
C:\Users\Administrator\Desktop\CET4-已有音频-1104.apkg
```

`.apkg` 是可导入、可转移的 Anki 卡组包，不是直接阅读的文档。卡组导入后，日常学习记录保存在 Anki 自己的资料库中，不会写回这个 `.apkg` 文件。

## 每天怎样学习

1. 双击桌面的 `Anki`；
2. 打开 `英语四级乱序词汇 · 已生成音频`；
3. 点击“开始学习”；
4. 先听发音并回忆词义，再点击“显示答案”；
5. 根据实际回忆情况选择：
   - “再来”：没有想起来或答错；
   - “困难”：想起来了，但很吃力；
   - “良好”：正常回忆出来，日常优先选这个；
   - “简单”：立刻想起而且非常熟悉。

Anki 会按记忆情况自动安排下一次复习。当前每天先学习最多 30 个新词，再完成当天到期的复习卡。

如果需要重播当前单词，按键盘 `R`，或点击卡片上的重播音频按钮。

## 以后怎样导入新的 `.apkg`

1. 双击 `.apkg` 文件，或在 Anki 中选择“文件 → 导入”；
2. 新卡组保持“不导入学习进度”；
3. 确认卡组名称和卡片数量后点击“导入”；
4. 导入完成后抽查一个单词的正面、释义和发音。

如果是本项目对同一批词条重新导出的更新卡组，导入前先保留原 `.apkg`，并在导入结果页检查“新增”和“更新”的数量，避免误导入重复卡组。

## 数据位置与备份

Anki 当前本地资料位于：

```text
C:\Users\Administrator\AppData\Roaming\Anki2\账户 1\
```

其中：

- `collection.anki2` 保存卡片和学习进度；
- `collection.media\` 保存导入的音频，当前有 1104 个 WAV；
- 桌面的 `.apkg` 可作为本次 1104 张卡片的独立导入备份。

不要在 Anki 运行时手工移动或删除资料库文件。需要换电脑时，优先从 Anki 导出包含媒体的卡组，或使用 AnkiWeb 同步；启用 AnkiWeb 需要用户自己的账号，当前没有代为注册或登录。
