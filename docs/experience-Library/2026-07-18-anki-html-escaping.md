# Anki 字段中的教材尖括号标记被隐藏

## 现象

真实词表导出时，`genanki` 警告释义包含无效 HTML 标签。教材使用的 `<古>`、`<口>`、`<俚>` 等语体标记在 Anki 中可能被当作网页标签而不显示。

## 根本原因

Anki 卡片字段支持 HTML。直接把 PDF 原文写入字段时，尖括号内容会进入 HTML 解析流程，而不是按普通文字展示。

## 解决办法

导出前对单词、音标和释义调用 `html.escape()`。卡片模板仍保留自身 HTML，用户词条内容则按纯文字安全显示。

## 验证方式

自动测试解包 `.apkg`，读取 `collection.anki2` 的 `notes.flds`，确认 `<古>` 被保存为 `&lt;古&gt;`，且不再包含原始标签。

## 防复发规则

1. PDF、人工编辑或外部导入的字段进入 Anki 前必须进行 HTML 转义。
2. 不对 `[sound:文件名]` 和受控卡片模板做相同转义。
3. Anki 测试样本必须包含至少一个尖括号教材标记。
