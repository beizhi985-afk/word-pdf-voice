# 开发经验库

本目录记录已经解决、可能再次出现，或对后续开发有复用价值的问题。

## 何时新增记录

- 同一个问题排查超过一次；
- 问题与 PDF 版式、OCR、字符编码、音标或语音错读有关；
- 修复依赖特定版本、配置或平台差异；
- 问题可能导致数据错配、源文件损坏或隐私风险；
- 解决方法对未来功能具有通用价值。

## 建议模板

```markdown
# 问题标题

## 现象

## 影响范围

## 根本原因

## 解决办法

## 验证方式

## 防复发规则

## 相关文件或版本
```

文件名建议使用日期和简短主题，例如 `2026-07-17-pdf-reading-order.md`。新增记录后必须更新 `docs/index.md`。
