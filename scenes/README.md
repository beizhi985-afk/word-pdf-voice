# 前端场景与审美文件

本目录用于描述和保存产品的界面场景、交互状态与视觉参考。

首版重点场景预计包括：

1. 空状态与 PDF 拖入。
2. 文档分析和处理进度。
3. 词条预览、低置信度提示和人工修正。
4. 单词与例句的试听、切换声音和语速。
5. 批量生成状态、失败重试和导出。
6. 离线模式、云端模式和费用/隐私提示。

## 审美原则

- 面向学习者，安静、清晰，减少无关装饰。
- 优先展示“原文、结构化词条、语音状态”三者关系。
- 错误和低置信度结果必须明显，但不打断正常校对流程。
- 批量任务应展示总量、已完成、失败和剩余时间。
- 关键操作使用直白中文，避免要求用户理解 OCR、TTS 等技术术语。

后续的线框图、页面截图、视觉参考和交互说明均放在本目录。
