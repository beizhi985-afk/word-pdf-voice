# assets 使用说明

本目录用于保存项目所需的资源文件，例如：

- 脱敏后的 PDF 测试页或截图；
- 界面图片、图标和品牌资源；
- 经许可使用的短音频样例；
- 测试所需的小型静态数据。

## 规则

1. 不提交未经许可传播的完整教材或受版权保护的 PDF。
2. 优先保存最小、脱敏、可复现的测试样本。
3. 大型模型、批量生成音频和临时输出不进入 Git；它们应保存在被 `.gitignore` 排除的位置。
4. 新资源应使用可理解的名称，并在相关文档中说明来源、用途和授权情况。
5. 密钥、令牌、个人信息和私有文档不得放入本目录。
