# v0.4.0 原创贴纸与应用图标

本目录保存公开版内置的五枚原创透明 PNG，以及应用小书图标：

- `chibi-student.png`：抱着单词卡片的学习者；
- `cozy-cloud-cat.png`：在云朵上读书的小猫；
- `moon-rabbit-study.png`：抱着词卡的小月兔；
- `clever-fox-study.png`：戴圆眼镜、读书和整理图形积木的小橘狐；
- `sprout-robot-study.png`：照料学习嫩芽的小机器人；
- `app-icon.png` / `app-icon.ico`：程序窗口、可执行文件和桌面快捷方式使用的可爱小书。

## 来源与版权边界

以上五枚贴纸和小书图标均为本项目通过生成式图像工具制作的原创学习主题资源，不使用商业动漫人物、品牌标识或用户教材内容。月兔、橘狐和机器人于 2026-07-19 使用内置图像生成工具创建，并通过纯色背景移除流程生成透明 PNG。

v0.3.1 曾包含的五枚商业动漫角色同人贴纸已在 v0.4.0 从公开仓库和安装包删除。用户如果在个人电脑中加入自己的图片，文件只保存在 `%LOCALAPPDATA%\WordPdfVoice\v0.2\custom-stickers\`，不会进入 GitHub 或正式安装包。

## 使用规则

1. 程序每次启动从五枚内置原创贴纸和用户本地贴纸中选择一枚，并避免连续重复。
2. “＋我的贴纸”支持 PNG、JPG、JPEG 和 WebP；删除操作只作用于用户自己的本地贴纸。
3. PyInstaller 将本目录 PNG 和 ICO 收集到发布包的 `assets/ui/`，并把 ICO 写入可执行文件。
4. 新增内置资源必须记录来源、保留透明背景，并通过资源加载、透明通道和版权边界测试。
