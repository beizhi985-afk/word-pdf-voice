# v0.3.0 原创界面贴纸

本目录保存第三版桌面界面随机展示的三枚透明 PNG：

- `chibi-student.png`：抱着单词卡片的学习者；
- `headphone-dino.png`：戴耳机练听力的小恐龙；
- `cozy-cloud-cat.png`：在云朵上读书的小猫。

## 来源与版权边界

三枚贴纸均于 2026-07-18 为本项目通过生成式图像工具原创生成，随后去除纯色背景并检查透明边缘。设计提示强调“原创、可爱、温暖、学习主题、透明背景贴纸”，没有使用蜡笔小新、柯南等商业角色的姓名、形象或临摹要求。

这些资源用于本项目界面、截图和发布包。不要把它们标记成任何现有动漫作品的官方素材，也不要替换为来源不明的网络贴纸。

## 使用规则

1. 程序每次启动从三枚贴纸中选择一枚，并避免与上一次连续重复。
2. PyInstaller 将本目录 PNG 收集到发布包的 `assets/ui/`。
3. 新增或替换贴纸时，必须保留透明背景、记录来源，并运行资源加载与透明通道测试。
