# v0.3.1 界面贴纸与应用图标

本目录保存桌面界面随机展示的七枚透明 PNG，以及应用小书图标：

- `crayon-shinnosuke.png`：拿着字母卡的小新；
- `crayon-kazama.png`：抱着蓝色单词书的风间彻；
- `crayon-masao.png`：拿着单词卡的正南；
- `crayon-bochan.png`：拿着紫色单词卡的阿呆；
- `crayon-nene.png`：拿着图卡的妮妮；
- `chibi-student.png`：抱着单词卡片的学习者；
- `cozy-cloud-cat.png`：在云朵上读书的小猫；
- `app-icon.png` / `app-icon.ico`：程序窗口、可执行文件和桌面快捷方式使用的可爱小书。

## 来源与版权边界

原创学习者、云朵读书猫和小书图标于 2026-07-18 为本项目通过生成式图像工具生成。五枚角色贴纸是用户要求的个人学习界面同人化视觉资源；风间、正南、阿呆、妮妮使用用户提供的截图作为外观参考，小新按文字要求生成。参考截图不进入仓库，只提交去除纯色背景后的学习主题贴纸。

五名角色及《蜡笔小新》相关权利归原权利方所有；这些贴纸不是官方素材，也不得被标记为官方授权。仓库只把它们作为用户指定的个人学习软件界面资源使用。

## 使用规则

1. 程序每次启动从七枚贴纸中选择一枚，并避免与上一次连续重复。
2. PyInstaller 将本目录 PNG 和 ICO 收集到发布包的 `assets/ui/`，并把 ICO 写入可执行文件。
3. 新增或替换贴纸时，必须保留透明背景、记录来源，并运行资源加载与透明通道测试。
