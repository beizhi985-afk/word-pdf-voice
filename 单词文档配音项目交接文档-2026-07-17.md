## 3. 运行入口

普通使用直接运行：

```text
dist\WordPdfVoice\WordPdfVoice.exe
```

开发环境：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\setup.ps1
.\.venv\Scripts\python.exe .\scripts\run_app.py
```

构建便携版：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build.ps1
```

## 4. 本地数据与隐私

- 项目数据库、生成音频和导出卡组保存在 `%LOCALAPPDATA%\WordPdfVoice\projects\`。
- eSpeak 兼容运行时保存在 `%LOCALAPPDATA%\WordPdfVoice\runtime\`。
- `models/`、`dist/`、`output/`、`.venv/`、PDF、WAV 和 `.apkg` 均不提交 Git。
- 不在仓库、文档或日志中保存 GitHub Token、API 密钥或个人文档。
