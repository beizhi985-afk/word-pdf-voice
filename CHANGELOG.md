# 更新日志

本文件记录用户可感知的功能变化、重要修复和版本范围调整。

## [0.6.0] - 2026-07-22

### 智能文档解析

- 新增分级解析：规则四列表格继续使用原有快速提取，双栏中英笔记自动改用文字坐标与版面顺序解析。
- 复杂文档按每页左栏、右栏的阅读顺序生成候选词条，保存来源页码、坐标、识别方式和置信度。
- 低密度说明页自动跳过；没有有效文字层时给出 OCR 扩展提示，不再长时间停留后只显示笼统失败。

### 导入确认与可恢复性

- 新增复杂文档导入预览，可勾选、编辑、删除、合并候选行，并可只保留高置信度内容；确认前不写入词库。
- 文档分析增加安全取消入口；取消分析或关闭预览后，已有项目数据库保持不变。
- 数据库结构升级到 v0.6.0，新增置信度、来源坐标和识别方式字段；迁移前自动创建 SQLite 完整性备份。

### 兼容与验证

- 《雅思阅读笔记.pdf》真实回归得到 1772 条候选，自动跳过第 1 页说明，处理时间约 33 秒。
- 四级词汇 PDF 真实回归仍为 107 页、4450 条、序号 1—4450 连续，旧表格解析没有回退。
- 源码、构建配置、输出目录和可执行文件统一升级为 v0.6.0；37 项自动测试运行完成且结果为 OK（其中 3 项按环境条件跳过），最终 EXE 对两份真实 PDF 的冻结版提取门禁和多声音、多语速发音门禁全部通过。

## [0.5.0] - 2026-07-22

- 按“页面做减法”的方向重排主界面：紧凑标题栏、单行筛选区、四列表格和常驻底部学习播放器成为唯一主层级。

- 增加专注筛选和播放器选中联动测试，并为三枚传输按钮增加尺寸、类型和圆形主按钮对象回归检查。

### 兼容与发布

## [0.4.0] - 2026-07-19

### 连续学习

- 可从当前选中单词开始，按表格当前排序和筛选顺序自动播放后续单词。
- 单词重复次数可选 1—5 次，词间暂停可选 0—10 秒；播放进度显示当前位置、单词和重复次数。
- 内容可选“只听英文”或“英文＋中文释义”；中文使用 Windows 离线中文语音，没有中文语音时提示并继续播放英文。
- 增加“认识、模糊、不认识”三档标记；“重点复习”只显示模糊和不认识的词。
- 单词生成或播放失败时跳过当前项，不中断整个队列；“停止播放”和“停止全部”可中断连续任务。

### 贴纸与版权

- 增加“换贴纸”和“＋我的贴纸”，支持导入 PNG、JPG、JPEG、WebP，并可删除用户自己的本地贴纸。
- 公开版内置学习者、云朵读书猫、月兔、橘狐、学习机器人五枚原创透明贴纸。
- 删除 v0.3.1 的五枚商业动漫角色同人贴纸；用户自定义图片只保存在本机，不进入 GitHub 或安装包。

### 数据保护与兼容

- v0.3.x 数据首次打开时先备份，再自动增加 `learning_status`；原词条、人工修改、音频状态和缓存配置保持不变。
- 每天首次打开、重新导入和恢复前自动创建 SQLite 一致性备份，默认保留最近 12 份。
- “选择词汇 → 备份与恢复”支持立即备份、查看历史和恢复；备份与恢复结果都执行数据库完整性检查。
- 恢复前保存当前有效数据库，并通过临时数据库检查后原子替换，降低误覆盖风险。

### 发布

- 窗口标题、版本徽标、源码、构建目录和可执行文件升级为 v0.4.0。
- 构建前运行版本一致性门禁，版本不一致时停止生成正式包。
- 31 项自动测试与真实 107 页 PDF 回归全部通过；真实旧库保留 4450 条词和 2521 条音频，并创建 2 份保护性备份。
- 便携版真实英语 WAV 门禁、Windows 中文离线语音检测和实机界面验收通过；截图归档到 `assets/screenshots/v0.4.0-continuous-learning.png`。
- 正式便携版已部署到 Obsidian 仓库，桌面的通用入口和 v0.4.0 版本入口均指向新程序；旧版本仍保留用于回退。

## [0.1.0] - 2026-07-17
