# v0.1.1 归档规格

状态：已发布
发布日期：2026-07-18

## 版本目标

把目标英语四级词汇 PDF 转换为可逐词播放、可校对并可导出到 Anki 的本地学习资料，并修复便携版遗漏 `language_tags` 数据导致的发音失败。

## 主要能力

- 提取目标 PDF 的 107 页、4450 条四列表格。
- 搜索、异常筛选、人工编辑和逐词试听。
- 本地 Kokoro ONNX 发音、30 词样本和全部生成。
- SQLite 音频缓存与中断续做。
- 仅在 4450 条音频全部存在时导出 Anki `.apkg`。
- Windows 便携版内置语音模型，并在构建后生成真实 WAV 验证资源完整性。

## 数据与程序目录

- 程序：`dist\WordPdfVoice\WordPdfVoice.exe`
- 项目数据：`%LOCALAPPDATA%\WordPdfVoice\projects\`
- eSpeak 运行时：`%LOCALAPPDATA%\WordPdfVoice\runtime\`

## 已知限制

- 不能只查看已有音频词条。
- 不能提前导出部分 Anki 卡组。
- 只保证目标 PDF 和同类文本型四列表格。

后续能力见 [当前版本规格](current.md)。
